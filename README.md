🌤 Weather Dashboard App

A modern and responsive Weather Application built with React that allows users to search any city and get real-time weather information using an external API.

🚀 Live Demo

👉 https://weather-dashboard-one-nu.vercel.app

📸 Preview

✨ Features
🔍 Search weather by city name
🌡 Real-time temperature display
🌥 Weather condition icons (Clear, Rain, Clouds, Snow)
🎨 Dynamic background based on weather
⚡ Loading indicator for better UX
❌ Error handling for invalid cities
📱 Fully responsive design
🧠 Custom React Hook for API logic
🛠 Tech Stack
React.js (Hooks: useState, custom hooks)
JavaScript (ES6+)
CSS (CSS Variables + Responsive Design)
OpenWeather API
React Icons
